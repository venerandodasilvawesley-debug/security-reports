VUL-004 ✅ — Autenticação JWT (bank_api)
──────────────────────────────────────────────────────────────────
Arquivo criado: bank_api/app/auth.py
  • OAuth2PasswordBearer com JWT (HS256)
  • SECRET_KEY via variável de ambiente
  • Função get_current_user() com tratamento de JWTError

Rotas protegidas:
  • POST /clientes/    → requer Bearer token
  • GET  /clientes/    → requer Bearer token
  • POST /contas/      → requer Bearer token
  • GET  /contas/      → requer Bearer token
  • POST /transacoes/  → requer Bearer token

Dependências adicionadas:
  python-jose[cryptography], passlib[bcrypt]
